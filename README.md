# AWS Static Website Project

**Submitted by:** Dhanushree K.

## Project Overview
This project demonstrates hosting a **static website** on **Amazon S3** and distributing it via **CloudFront CDN** as part of AWS learning and certification.

The website features:
- A welcoming landing page with a calm pastel background
- Heading: "Welcome to my AWS Static Website"
- Creator's name: "Created by Dhanushree K."
- Inspiring quote: "Learn. Create. Grow!"

## Technologies Used
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront (Content Delivery Network)
- HTML & CSS

- **index.html**  
  The main entry point of the static website.  
  It contains the HTML structure for the landing page, including the heading, creator name, inspirational quote, and links to the CSS styles and assets.

  Contains all the styling for the website, such as layout, fonts, colors, alignment, and background image configuration to give the page a clean pastel look.
